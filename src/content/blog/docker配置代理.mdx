## docker pull 拉取镜像时设置代理

docker pull 的代理被 systemd 接管，所以需要设置 systemd 的代理

sudo vim /etc/systemd/system/docker.service.d/http-proxy.conf

```shell
[Service]
Environment="HTTP_PROXY=http://192.168.55.201:7890"
Environment="HTTPS_PROXY=http://192.168.55.201:7890"
# 重启docker
sudo systemctl daemon-reload
sudo systemctl restart docker
```
