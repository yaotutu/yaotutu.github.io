## 1. 介绍
基于PVE的打造家庭服务器，主要是为了实现家庭服务器的功能，如NAS、媒体服务器、下载服务器等。
* OMV
    * 担任Nas的功能，提供文件共享，照片备份，视频流媒体等功能。
* OpenWRT
    * 担任路由器的功能，提供家庭网络的管理，防火墙，VPN等功能。
* ubuntu server
    * 用于远程开发，编译，运行一些服务。
* Windows
    * 用于一些特殊软件的运行，如游戏，办公软件等。
